Current State

JSCover is based on the mature JSCoverage program (for differences with JSCoverage, please see the manual). It has re-used JSCoverage regression tests (see InstrumentAndHighlightRegressionTest) and is constantly adding more to improve the quality. JSCover has a suite of unit, integration and acceptance tests which give a fairly high degree of coverage (~95% line, ~90% branch). JavaScript tests are also being added (using Jasmine) and JSCover is used to check JavaScript coverage (see HtmlUnitJSTest).
