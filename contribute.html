Contribute

Please feel free to contribute - see the To-Do list and github issues. The difficulty level for contributing patches varies depending on the area.

Closure-compiler bugs can be posted there, but please notify us to avoid duplication. If the report is urgent it can be manually patched and included in JSCover.
